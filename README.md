# Frog Mod
### Design by xef5000 and Yamiko


A Hypixel Skyblock Utilities mod.

- Lots of Overlay/Tracker
- Original features
 
## Features

#### General
- Better Trade Menu
- Full NPC Price

#### Dungeons
- Custom death message
- Special 270/300 score message
- Show extra stats
- Watcher Ready/Cleared alert
- Terminal ESP

#### Mining
- Crystal Scanner (Current locations: Jungle Temple, Divan Mine, King/Queen, City, Bal, Throne, Corleone, Grotto, Xalx, Pete, Odawa)
- Worm/Scatha alert
- Unbreakable Cobble

#### Farming
- Milstone display
- Visitor Notifier
- 1x1x1 Crops HitBox

#### Combat
- Barbarian ESP
- Artifical Ferocity
